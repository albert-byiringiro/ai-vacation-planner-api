# An intelligent assistant that helps users plan trips end-to-end using conversational AI.


## Final Goal

A system that can:

- Suggest destinations
- Generate itineraries (Automatically create a travel plan for the user.)
- Fetch real-time info (maps, weather, pricing)
- Store conversation context
- Retrieve knowledge (RAG)
- Use tools (APIs, search, maps)
- Handle multimodal inputs (text, voice, images)
- Be evaluated, secured, and production-ready
